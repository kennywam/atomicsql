# AtomicSQL

A simple RDBMS implementation in TypeScript.

## Setup

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Build the project:

   ```bash
   pnpm build
   ```

3. Run the project:
   ```bash
   pnpm start
   ```

## Development

- Watch mode for development:

  ```bash
  pnpm dev
  ```

- Clean build artifacts:
  ```bash
  pnpm clean
  ```

## Project Structure

- `src/` - TypeScript source code
- `dist/` - Compiled JavaScript output
- `tsconfig.json` - TypeScript configuration
